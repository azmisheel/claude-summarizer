A web app for turning PDFs into something actually useful. Upload a document and get a summary, a set of flashcards, or a quiz without having to read the whole thing yourself.
Features (in progress)

PDF upload and management
Document summarization
Flashcard generation
Quiz generation
Chat with your document

Stack

Frontend: React, TypeScript, Vite, Tailwind CSS
Backend: Node.js, Express
Database: MongoDB Atlas
AI: Claude API by Anthropic

Status
Still in development. Core structure is set up — AI integration coming next.
